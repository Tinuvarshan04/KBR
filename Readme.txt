
# KBR Advertising Company Website  

This is a responsive website designed for an advertising company. The project showcases a clean and user-friendly interface, highlighting key services and portfolios. The website is developed using **HTML**, **CSS**, and **JavaScript**.  

## Features  

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices.  
- **Interactive UI**: Smooth navigation with JavaScript functionality.  
- **Modern Aesthetic**: Designed with a focus on visual appeal and user engagement.  

## Technologies Used  

- **HTML**: Structure and content of the website.  
- **CSS**: Styling and layout of the web pages.  
- **JavaScript**: Interactivity and dynamic features.  

## Installation and Usage  

1. Clone this repository:  
   ```bash  
   git clone https://github.com/Tinuvarshan04/KBR.git  
   ```  
2. Open the project directory:  
   ```bash  
   cd KBR  
   ```  
3. Open the `index.html` file in your browser to view the website.
